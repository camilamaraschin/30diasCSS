HTML

Linguagem de Marcação de Hipertexto, o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

CSS

Cascading Style Sheet ou Folha de Estilo em Cascatas, é necessário dizer que sua aplicação é utilizada na estilização de componentes escritos em linguagens de marcação.

Java Script

Java Script é uma linguagem de programação que permite a você implementar itens complexos em páginas web toda vez que uma página da web faz mais do que simplesmente mostrar a você informação estática
